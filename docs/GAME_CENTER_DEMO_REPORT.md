## 📡 API接口

### 🎮 游戏相关API

**1. 游戏列表API**
```
GET /api/games
返回: 所有系统的游戏列表
```

**2. 游戏详情API**
```
GET /api/game/{system}/{game_id}
返回: 特定游戏的详细信息
```

**3. 存档API**
```
GET /api/saves/{game_id}
返回: 游戏的存档信息
```

**4. 金手指API**
```
GET /api/cheats/{game_id}
返回: 游戏的金手指配置
```
